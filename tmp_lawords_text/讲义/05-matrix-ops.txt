Matrixaddition und Skalarmultiplikation

Theorem (Eigenschaften der Addition und
Skalarmultiplikation)
Seien A, B, C ∈ Rm×n und r , s ∈ R. Dann gilt:

 (i) A + B = B + A
(ii) (A + B) + C = A + (B + C )
(iii) A + 0 = A
(iv) r (A + B) = rA + rB
(v) (r + s)A = rA + sA
(vi) r (sA) = (rs)A

Matrixmultiplikation




Definition (Matrixmultiplikation)
Sei A ∈ Rm×k und B ∈ Rk×n . Dann definieren wir das
Matrixprodukt AB als die Matrix

                   AB = (Ab1 · · · Abn ) ∈ Rm×n ,

wobei b1 , ..., bn die Spalten von B sind.

der Spalten in A gleich der Anzahl der Zeilen in B ist.

Theorem (Rechenregeln Matrixmultiplikation)
Sei A ∈ Rm×n und B, C weitere Matrizen, sodass die
untenstehenden Summen und Produkte wohldefiniert sind.

 (i) A(BC ) = (AB)C
(ii) A(B + C ) = AB + AC
(iii) (B + C )A = BA + CA
(iv) c(AB) = (cA)B = A(cB) für jedes ∈ R
(v) Im A = A = AIn

Matrixtransposition




Definition (Matrixtransposition)
Die Transposition A⊤ ∈ Rn×m eine Matrix A ∈ Rm×n ist definiert
durch (A⊤ )ij = Aji für alle i ∈ 1, . . . , n und j = 1, . . . m.

Matrixtransposition


Theorem (Rechenregeln Matrixtransposition)
Sei A ∈ Rm×n und B, C weitere Matrizen, sodass die
untenstehenden Summen und Produkte wohldefiniert sind.

 (i) (A⊤ )⊤ = A
(ii) (A + B)⊤ = A⊤ + B ⊤
(iii) (cA)⊤ = cA⊤ für jedes c ∈ R,
(iv) (AB)⊤ = B ⊤ A⊤ .
