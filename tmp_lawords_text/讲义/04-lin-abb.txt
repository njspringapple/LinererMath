Matrixtransformation




 • Für A ∈ Rm×n transformiert das Matrix-Vektor-Produkt jeden
   Vektor x ∈ Rn in einen Vektor Ax ∈ Rm .

 • Durch die Transformation T (x) = Ax wird also eine
   Abbildung T : Rn 7→ Rm definiert.

Allgemeine lineare Abbildungen

Definition
Wir nennen eine Abbildung T : D → W linear, wenn

 1. T (x + y ) = T (x) + T (y ) für alle x, y ∈ D,
 2. T (cx) = cT (x) für alle c ∈ R, x ∈ D.

Allgemeine lineare Abbildungen

Definition
Wir nennen eine Abbildung T : D → W linear, wenn

 1. T (x + y ) = T (x) + T (y ) für alle x, y ∈ D,
 2. T (cx) = cT (x) für alle c ∈ R, x ∈ D.


Bemerkung
Falls T eine lineare Transformation ist, dann:

 1. T (0) = 0.
 2. T (c1 x1 + · · · + ck xk ) = c1 T (x1 ) + · · · + ck T (xk ).

Allgemeine lineare Abbildung




Theorem
Eine Abbildung T : Rn 7→ Rm ist genau dann linear, wenn es eine
Matrix A ∈ Rm×n gibt, so dass T (x) = Ax für alle x ∈ Rn .
Diese Matrix ist eindeutig.

Lineare Abbildungen in Funktionenräumen



Beispiel

  • Sei F = L2 ([0, 1]) die Menge aller Funktionen f : [0, 1] 7→ R
        R1
    mit 0 f 2 (x)dx < ∞.

  • Fixiere g ∈ F und definiere die Abbildung T : F 7→ R durch
             R1
    T (f ) = 0 g (x)f (x)dx.

  • Dann ist T linear.
