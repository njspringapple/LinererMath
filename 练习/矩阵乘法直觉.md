## 三、证明题 / Beweisaufgaben（共2题）

**16.**
证明：$\mathbf{y} = A\mathbf{x}$（列向量左乘）与 $\mathbf{y}^T = \mathbf{x}^T A^T$（行向量右乘）是等价的，即一个是另一个的转置形式。

Beweisen Sie: $\mathbf{y} = A\mathbf{x}$ (Spaltenvektor links-multipliziert) und $\mathbf{y}^T = \mathbf{x}^T A^T$ (Zeilenvektor rechts-multipliziert) sind äquivalent, d. h. die eine Form ist die Transponierte der anderen.

**提示 / Hinweis:** 对 $\mathbf{y} = A\mathbf{x}$ 两边取转置，利用 $(AB)^T = B^T A^T$。

### 计算题

**11.**
$\mathbf{a}^T\mathbf{b}$ 是"横 × 竖"，$(1 \times n)(n \times 1) = 1 \times 1$，结果是一个标量。$\mathbf{a}\mathbf{b}^T$ 是"竖 × 横"，$(n \times 1)(1 \times n) = n \times n$，结果是一个矩阵。

**12.**
(a) $A\mathbf{x} = \alpha\,\mathbf{a}_1 + \beta\,\mathbf{a}_2$

(b) 结果是 3 维列向量。输入 2 维，输出 3 维，这是升维（$m > n$）。

**13.**
$y_i = \mathbf{r}_i^T \mathbf{x} = \sum_{j=1}^{n} a_{ij}\, x_j$

之所以叫"批量"内积，是因为 $m$ 个行向量各自独立地与同一个 $\mathbf{x}$ 做内积，一次性产生 $m$ 个结果，相当于把 $m$ 次内积打包完成。

**14.**
(a) 批量内积视角：$y_j = \mathbf{x}^T \mathbf{c}_j = \sum_{i=1}^{m} x_i\, a_{ij}$（$\mathbf{x}$ 与 $A$ 的每一列做内积）

(b) 加权和视角：$\mathbf{y}^T = x_1 \mathbf{r}_1^T + x_2 \mathbf{r}_2^T + \cdots + x_m \mathbf{r}_m^T = \sum_{i=1}^m x_i \mathbf{r}_i^T$

### 证明题

**15.**
线性组合视角：$A\mathbf{x} = \sum_{j=1}^n x_j \mathbf{a}_j$，其第 $i$ 个分量为 $\sum_{j=1}^n a_{ij} x_j$。

批量内积视角：第 $i$ 个分量为 $\mathbf{r}_i^T \mathbf{x} = \sum_{j=1}^n a_{ij} x_j$。

两者的第 $i$ 个分量表达式完全相同，故两种视角等价。$\blacksquare$

**16.**
由 $\mathbf{y} = A\mathbf{x}$，两边取转置：

$$\mathbf{y}^T = (A\mathbf{x})^T = \mathbf{x}^T A^T$$

这正是行向量右乘的形式。反过来，对 $\mathbf{y}^T = \mathbf{x}^T A^T$ 两边取转置可得 $\mathbf{y} = A\mathbf{x}$。因此两种表达等价。$\blacksquare$
