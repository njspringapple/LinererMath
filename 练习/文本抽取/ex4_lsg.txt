Recap - Invertierbarkeit
  1. Das Matrix Produkt verhält sich ähnlich (Theorem 2.1.8), unterscheidet sich aber grund-
     legend in zwei Eigenschaften: es ist nicht kommutativ (d.h. AB ̸= BA i.A.) und besitzt
     im Allgemeinen kein Inverses (d.h. ∄B s.d. AB = 1 i.A.)

  2. Dass ein Inverses (d.h. A−1 ) existiert, ist (unter anderem) äquivalent zur eindeutigen
     Lösbarkeit des LGS Ax = b für beliebige b (Theorem 2.2.15)

  3. Schränken wir uns auf invertierbare Matrizen ein, so gelten analoge Rechenregeln wie
     (AB)−1 = B −1 A−1 (Theorem 2.2.3)

  4. Matrixaddition entspricht der Addition von linearen Abbildungen. Matrixmultiplikation
     entspricht der Komposition von linearen Abbildungen: seien A, B darstellende Matrizen
     von f, g, dann ist AB die darstellende Matrix von f ◦ g = f (g(x)). Matrixinvertierung
     entspricht der Invertierung von linearen Funktionen.


Fingerübungen
  1. Wenn das LGS Ax = b eine Lösung besitzt und B eine invertierbare Matrix ist, besitzt
     dann das LGS (AB)x = b eine Lösung?


Übungsaufgaben
Aufgabe 1 (Das Inverse einer linearen Abbildung ist linear)
Sei f eine lineare Abbildung. Zeige (ohne Nutzung von f −1 (x) = A−1 x mit A darstellender
Matrix von f ): Wenn f eine invertierbare Abbildung ist, d.h. es existiert eine Abbildung f −1
mit f ◦ f −1 (x) = x = f −1 ◦ f (x) für alle x, dann ist f −1 linear.



Lösungsskizze zu Aufgabe 1
Sei f linear und invertierbar. Für f −1 gilt unter Nutzung der Linearität von f

  f −1 (x + y) = f −1 (f [f −1 (x)] + f [f −1 (x)]) = f −1 (f [f −1 (x) + f −1 (x)]) = f −1 (x) + f −1 (x)

und
                    f −1 (cx) = f −1 (cf [f −1 (x)]) = f −1 (f [cf −1 (x)]) = cf −1 (x).
Somit ist f −1 eine lineare Abbildung.
Aufgabe 2 (Im Allgemeinen besitzen Matrizen kein Inverses)
Seien 0n×n ̸= A, B zwei von Null verschiedene n × n-Matrizen mit AB = 0.
Zeige: A besitzt kein (links-)Inverses bezüglich des Matrix Produkts, d.h. es existiert keine
Matrix C mit CA = 1n×n .


Lösungsskizze zu Aufgabe 2
Beweis durch Widerspruch: Wir nehmen an, dass eine Matrix C existiert mit CA = 1n×n .
Dann gilt (CA)B = 1n×n B = B ̸= 0, aber C(AB) = C · 0 = 0. Das ist ein Widerspruch, da für
Matrixmultiplikation (CA)B = C(AB) gilt. Also kann es keine solche Matrix C geben.

Aufgabe 3 (Invertierbarkeit und bijektive Abbildungen)
Sei f : Rn → Rn eine lineare Abbildung.

   • Wir nennen f injektiv, genau dann wenn für alle x ∈ Rn mit f (x) = 0 schon x = 0
     folgt.1

   • Wir nennen f surjektiv, genau dann wenn für alle y ∈ Rn ein x ∈ Rn existiert mit
     f (x) = y.

   • f ist bijektiv, wenn f sowohl injektiv als auch surjektiv ist. Eine Funktion ist invertierbar
     genau dann, wenn sie bijektiv ist.

Zeige: f ist invertierbar ⇔ f ist injektiv ⇔ f ist surjektiv.
Hinweis: Nutze die Darstellung von f über eine Matrix A. Was bedeutet es für A, wenn f
injektiv bzw. surjektiv ist? Nutze dann Theorem 2.2.15.


Lösungsskizze zu Aufgabe 3
Die Aussagen (f invertierbar =⇒ f injektiv) und (f invertierbar =⇒ f surjektiv) folgen
direkt aus (f invertierbar ⇔ f bijektiv). Es bleibt zu zeigen: (f injektiv =⇒ f invertierbar)
und (f surjektiv =⇒ f invertierbar).
Sei A die darstellende Matrix von f . Wenn A invertierbar ist, dann ist auch f invertierbar,
wobei A−1 die darstellende Matrix von f −1 ist.
Sei f injektiv. Aus der obigen Definition folgt, dass das LGS Ax = 0 nur die triviale Lösung
x = 0 hat. Laut Theorem 2.2.15 ist A dann invertierbar und somit ist f invertierbar.
Sei f surjektiv. Aus der obigen Definition folgt, dass das LGS Ax = y für jedes y ∈ Rn eine
Lösung hat. Laut Theorem 2.2.15 ist A dann invertierbar und somit ist f invertierbar.




   1
    Die aus Analysis bekannte Definition lautet: f injektiv ⇔ ∀x1 , x2 : (f (x1 ) = f (x2 ) =⇒ x1 = x2 ), d.h. für
jedes Element y im Wertebereich gibt es höchstens ein x im Definitionsbereich, sodass f (x) = y. Für lineare
Abbildungen ist die obige Definition äquivalent:
   – f injektiv =⇒ ∀x : (f (x) = 0 =⇒ x = 0) gilt, da für lineare Abbildungen immer f (0) = 0 gilt. Wenn
     f gleichzeitig injektiv ist, kann nur 0 auf 0 abgebildet werden.
   – ∀x : (f (x) = 0 =⇒ x = 0) =⇒ f injektiv: Beweis durch Widerspruch. Sei f nicht injektiv und
     ∀x : (f (x) = 0 =⇒ x = 0). Da f nicht injektiv, gibt es x1 ̸= x2 , sodass f (x1 ) = f (x2 ). Aus der
     Linearität von f folgt f (x1 − x2 ) = f (x1 ) − f (x2 ) = 0, aber x1 − x2 ̸= 0, was ein Widerspruch zu
     ∀x : (f (x) = 0 =⇒ x = 0) ist.
