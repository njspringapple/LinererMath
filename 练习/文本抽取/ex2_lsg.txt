Recap - Lineare Unabhängigkeit und Abbildungen
  1. Wir können jedes Lineare Gleichungssystem (LGS) schreiben als Ax = b wobei A eine
     Matrix und b ein Vektor ist.

  2. Das Matrix-Vektor Produkt Ax ist linear in x (Theorem 1.7.5).

  3. Über den Spann können wir den Lösungsraum eines LGS beschreiben (Theorem 1.7.6)

  4. Haben wir eine Lösung x0 mit Ax0 = b gefunden, dann lässt sich jede weitere Lösung
     schreiben als x0 + x wobei Ax = 0 gilt. (Theorem 1.8.6)

  5. Die Menge aller Ax = 0 trägt eine lineare Struktur. Das heißt wenn Ax1 = Ax2 = 0 gilt
     so ist auch A(ax1 + bx2 ) = 0. Es gilt sogar allgemeiner...

  6. ...jede Lösung lässt sich als Linearkombination von endlich vielen und fixen xi schreiben.
     Mathematisch ausgedrückt, {x : Ax = 0} = span{x1 , ..., xn }. (Theorem 1.8.3)

  7. Lineare Unabhängigkeit wird das Kernkonzept um die Anzahl n der Vektoren im Spann
     span{x1 , ..., xn } möglichst gering zu halten. (Theorem 1.9.2 und Aufgabe 2.)


Fingerübungen
  1. Wenn x eine Lösung für Ax = 0 ist und λ ∈ R, ist dann y = λx auch eine Lösung (d.h.
     Ay = 0)? Gilt dasselbe auch für Ax = b (d.h. Ay = b)? (Theorem 1.7.5)

  2. Wenn es nur endlich viele x1 , ..., xn gibt, die das LGS Ax = 0 lösen, was ist dann n?
     (Theorem 1.3.1 oder Fingerübung 1.)

  3. Kann es 2024 linear unabhängige Vektoren im R2023 geben? (Theorem 1.9.5)



Lösungsskizze zu Aufgabe 0
1. Für Ax = 0 gilt mit λ ∈ R laut Theorem 1.7.5.: A(λx) = λ(Ax) = λ0, somit ist auchλx
eine Lösung des LGS.
Für Ax = b gilt A(λx) = λb ̸= b im Allgemeinen, also ist λx i.A. keine Lösung. 2. Wenn es
ein x ̸= 0 mit Ax = 0 gibt (wir nennen das eine nicht-triviale Lösung, die triviale Lösung ist
x = 0), dann ist λx laut 1. ebenfalls eine Lösung, somit gibt es unendlich viele Lösungen, was
ein Widerspruch zur Aufgabenstellung ist. Also muss x = 0 gelten, somit n = 1.
Alternative: Aus Theorem 1.3.1 wissen wir, dass jedes LGS 0, 1 oder ∞ viele Lösungen hat,
somit kann es hier nur genau eine Lösung geben (denn x = 0 ist immer eine Lösung von
Ax = 0).

Lösungsskizze zu Aufgabe 1
Zu (a):
Die Aussage ist korrekt: Sei x ∈ span(M ). Per Definition gilt dann, dass a1 , ..., ak ∈ R existieren
mit
                                      x = a1 m1 + ..ak mk ,
d.h. x ist eine Linearkombination von Vektoren aus M . Ein Vektor y ist genau dann in span(N ),
wenn sich y als Linearkombination von Vektoren aus N schreiben lässt. Da M ⊆ N gilt
1 , ...,k ∈ N und somit x ∈ span(N ).
Zu (b):
Die Aussage ist falsch: Es reicht ein Gegenbeispiel: Idee: 0 ist in jedem Spann enthalten - auch
wenn die zugrundeliegende Menge an Vektoren, die den Spann erzeugt, nicht die 0 enthält.
In anderen Worten, M = {0}, N = {1} ⊂ R1 ist ein Gegenbeispiel, denn M ist nicht in N
enthalten aber span(M ) = {0} ⊂ span(N ) = R.
Alternative: für alle a ̸= b mit a, b, ∈ R, a ̸= 0, b ̸= 0 gilt span{a} = span{b} = R, aber weder
{a} ⊆ {b} noch {b} ⊆ {a}.
Zu (c):
Die Aussage ist falsch: Es reicht wieder ein Gegenbeispiel: Sei

                                      M = {e1 }, N = {e2 } ⊂ R2 .

Dann ist span(M ∪ N ) = R2 , aber span(M ) = {(x, 0) : x ∈ R} und span(N ) = {(0, x) : x ∈ R}
und somit span(M ) ∪ span(N ) = {(x1 , x2 ) : x1 = 0 ∨ x2 = 0} ̸= R2 .
Aufgabe 2 (Eine maximal linear unabhängige Teilmenge existiert)

Seien x1 , ..., xk ∈ Rn Vektoren. Zeige: es existieren 1 ≤ i1 , ..., il ≤ k, sodass {xi1 , ..., xil } linear
unabhängig ist und
                               span{xi1 , ..., xil } = span{x1 , ..., xk }.
In anderen Worten: jede Menge an Vektoren besitzt eine maximal linear unabhängige Teilmenge.
Hinweis: Theorem 1.9.2.


Lösungsskizze zu Aufgabe 2
Idee: wenn die Menge {x1 , ..., xk } linear abhängig ist, dann können wir einen Vektor daraus ent-
fernen, ohne den Spann zu verändern (folgt aus Th. 1.9.2). Das kann man so lange wiederholen,
bis die resultierende Menge linear unabhängig ist. Konkreter: Wir starten bei x1 und entfernen
es aus der Menge, wenn sich dadurch der Spann nicht verändert. Aus der resultierenden Menge
entfernen wir x2 , wenn sich dadurch der Spann nicht verändert etc.

    Aus Fall 1 und Fall 2 folgt nun die Behauptung.
    Alternativ funktioniert das ganze auch ohne Verwendung des Spanns: Sei x ∈ L. Dann
    ist cx (c ̸= 0) ebenfalls eine Lösung des LGS. Also ist cx ∈ L und somit cL ⊆ L. Noch
    zu zeigen: L ⊆ cL. Sei x ∈ L. Dann ist auch 1/cx (c ̸= 0) eine Lösung des LGS. Da also
    1/cx ∈ L, folgt x ∈ cL.

  2. Zu zeigen: x1 + L = x2 + L

    Fall 1: L = {0}
    Beweis durch Widerspruch. Nehme an, dass x1 ̸= x2 . Dann gilt

                                  Ax1 = b = Ax2 und x1 − x2 ̸= 0
                            ⇒ A(x1 − x2 ) = Ax1 − Ax2 = b − b = 0
                                                ⇒ 0 ̸= x1 − x2 ∈ L

    Das ist aber ein Widerspruch zu L = {0} ⇒ x1 = x2 und Ax = b hat hier nur genau
    eine Lösung.

    Fall 2: L ⊋ {0}.

    Wir zeigen zuerst, dass x1 + L ⊆ x2 + L. Sei x ∈ x1 + L, d.h. es existiert ein y ∈ L,
    sodass x = x1 + y. Zu zeigen: x ∈ x2 + L, d.h. es existiert ein ỹ ∈ L, sodass x = x2 + ỹ.
    Es gilt
                     x = x1 + x2 − x2 + y = x2 + (x1 − x2 + y) = x2 + ỹ
    mit ỹ = x1 − x2 + y. Es gilt ỹ ∈ L, da

                       A(x1 − x2 + y) = Ax1 − Ax2 + Ay = b − b − 0 = 0.

    Alternativ kann man ähnlich zeigen, dass x1 ∈ x2 + L (da x1 = x2 + (x1 − x2 )), woraus
    x1 + L ⊆ x2 + L folgt.

    Die andere Inklusion: x1 + L ⊇ x2 + L folgt analog.

    Aus Fall 1 und Fall 2 folgt nun die Behauptung.
