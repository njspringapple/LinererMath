Recap - Lineare Abbildungen und Matrixoperationen
  1. Die charakterisierende Eigenschaft von LGS, d.h. von Matrizen, ist, dass sie lineare Ab-
     bildungen definieren. Mathematisch ausgedrückt: jede Matrix definiert eine (eindeutige)
     lineare Abbildung und umgekehrt. (Theorem 1.10.2)
  2. In anderen Worten: das Studium von LGS ist das Studium von linearen Abbildungen.
  3. Wie für die reelen Zahlen können wir auch Matrizen addieren und multiplizieren
  4. Matrizen zu addieren hat alle analogen Eigenschaften wie Zahlen zu addieren (Theorem
     2.1.3)


Fingerübungen
  1. Wenn zwei lineare Abbildungen dieselbe darstellende Matrix haben, sind sie dann gleich?
     (Theorem 1.10.2)
  2. Wenn die Matrix A eine lineare Abbildung f darstellt und Ax = 0 gilt, was ist dann
     f (x)?
  3. Für Matrizen A, B, s.d. AT + B definiert ist, was ist (AT + B)T ? (Theorem 2.1.15)

Aufgabe 1 (Die Menge der Linearen Abbildungen ist linear)
Seien f, g : Rn → Rm lineare Abbildungen und a, b ∈ R. Zeige, dass dann auch

                                         h = af + bg : Rn → Rm

definiert durch h(x) = af (x) + bg(x) linear ist.


Lösungsskizze zu Aufgabe 1
Zu zeigen: h(x + y) = h(x) + h(y) und h(c x) = c h(x) für alle x, y ∈ Rn , c ∈ R. Es gilt
              Def. von h                                    f, g linear
   h(x + y)     =     af (x + y) + bg(x + y) = a(f (x) + f (y) + b(g(x) + g(y))
              = af (x) + bg(x) + af (y) + bg(y) = h(x) + h(y),
              Def. von h                      f, g linear
      h(cx)      =         af (cx) + bg(cx)       =          acf (x) + bcg(x) = c(af (x) + bg(x)) = c h(x).

Somit ist h eine lineare Abbildung.
