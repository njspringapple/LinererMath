Problem 8.1: (3.4 #13.(a,b,d) Introduction to Linear Algebra: Strang) Ex­
plain why these are all false:
a) The complete solution is any linear combination of x p and xn .

b) The system Ax = b has at most one particular solution.

c) If A is invertible there is no solution xn in the nullspace.

Solution:
a) The coefﬁcient of x p must be one.

b) If xn ∈ N( A) is in the nullspace of A and x p is one particular solution,
   then x p + xn is also a particular solution.

c) There’s always xn = 0.

Problem 8.3:    (3.4 #36.) Suppose Ax = b and Cx = b have the same
(complete) solutions for every b. Is it true that A = C?

Solution: Yes. In order to check that A = C as matrices, it is enough
to check that Ay = Cy for all vectors y of the correct size (or just for
the standard basis vectors, since multiplication by them “picks out the
columns”). So let y be any vector of the correct size, and set b = Ay. Then
y is certainly a solution to Ax = b, and so by our hypothesis must also be
a solution to Cx = b; in other words, Cy = b = Ay.
