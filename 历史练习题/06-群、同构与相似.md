# 群、同构与相似

## 相关公式
| 名称 | 公式 | 应用场景 |
|---|---|---|
| 子群判据 / Untergruppenkriterium | $e\in G,\;ab\in G,\;a^{-1}\in G$ | 证明矩阵集合在乘法下是子群。 |
| 同构 / Isomorphismus | $\varphi(xy)=\varphi(x)\varphi(y)$ 且双射 | 把矩阵乘法结构化成熟悉的数乘或整数加法。 |
| 相似不变量 / Ähnlichkeitsinvariante | $B=S^{-1}AS$ | 证明多项式方程、可逆性、幂、对角线量等在相似下保持。 |
