# 答案与解析

## 一、选择题（单选）答案

1. B（解析：矩阵乘法不满足交换律，即通常情况下AB≠BA。矩阵乘法满足结合律(AB)C=A(BC)和分配律A(B+C)=AB+AC，(B+C)A=BA+CA，且零矩阵是乘法的吸收元素。）
    
2. B（解析：矩阵乘法要求左矩阵的列数等于右矩阵的行数。A是3×3，B是3×2，C是2×3。对于BA，B的列数是2，而A的行数是3，不满足条件，因此BA无法计算。）
    
3. B（解析：根据矩阵可逆的等价条件，det(A)=0当且仅当A不可逆（奇异矩阵）。）
    
4. B（解析：若矩阵A满足A^2=A，则A被称为幂等矩阵。单位矩阵是一种特殊的幂等矩阵，但不是所有幂等矩阵都是单位矩阵。）
    
5. A（解析：通过初等行变换中的行交换和行相加操作，矩阵的行列式值可能会改变。但是，最终上三角矩阵的行列式等于对角线元素的乘积，即2×(-3)×5 = -30。因为行交换会改变行列式的符号，而行相加不改变行列式的值，所以原矩阵的行列式值取决于行交换的次数。如果有偶数次行交换，则结果是-30；如果有奇数次行交换，则结果是30。但题目只说"通过初等行变换"，而不是"只通过行相加操作"，因此选A。）
    

## 二、选择题（多选）答案

1. A、B（解析：
    
    - A正确：将矩阵的一行乘以非零常数k，行列式值会乘以k
    - B正确：交换矩阵的两行，行列式值会变为原来的相反数
    - C错误：将矩阵的一行的k倍加到另一行不改变行列式值
    - D错误：矩阵转置不改变行列式值，即det(A) = det(A^T)）
2. A、B、C、D（解析：这些都是可逆矩阵的基本性质。A^T可逆且(A^T)^(-1) = (A^(-1))^T；如果A和B都可逆，则(AB)^(-1) = B^(-1)A^(-1)；矩阵可逆等价于det(A)≠0；矩阵可逆等价于其列向量线性无关。）
    
3. A、B、C、D（解析：这些都是矩阵可逆的等价条件，根据矩阵可逆性的等价命题，这些条件互相等价。）

## 五、证明题答案

1. 证明：  
    我们用两种方法证明det(AB) = det(A)·det(B)：
    
    方法一：利用初等矩阵的性质
    
    任何可逆矩阵A都可以表示为有限个初等矩阵的乘积：A = E₁E₂...Eₖ  
    对于每个初等矩阵Eᵢ，有det(EᵢB) = det(Eᵢ)·det(B)
    
    依次应用这个性质：  
    $det(AB) = det(E₁E₂...EₖB) = det(E₁)·det(E₂...EₖB) = ... = det(E₁)·det(E₂)·...·det(Eₖ)·det(B) = det(E₁E₂...Eₖ)·det(B) = det(A)·det(B)$
    
    方法二：利用行列式的定义和性质
    
    我们可以证明，对于任何矩阵A和B，即使A或B不可逆，det(AB) = det(A)·det(B)也成立。
    
    如果A或B不可逆，则det(A) = 0或det(B) = 0，此时det(AB) = 0 = det(A)·det(B)
    
    如果A和B都可逆，则AB也可逆，且(AB)⁻¹ = B⁻¹A⁻¹
    
    利用行列式的余子式展开定义，可以直接验证对于2×2和3×3矩阵，det(AB) = det(A)·det(B)成立
    
    对于更高阶矩阵，可以使用数学归纳法证明。
    
2. 证明：  
    假设矩阵A的第i行和第j行成比例，即存在非零常数k，使得A的第i行 = k×(A的第j行)
    
    考虑计算det(A)：当我们沿着任意行或列展开行列式时，需要计算所有可能的排列组合。
    
    在每一种情况下，都需要从每行选择一个元素，然后计算这些元素的乘积（考虑适当的符号）。
    
    由于第i行和第j行成比例，任何包含第i行和第j行元素的排列组合都会出现两次，一次从第i行选择，一次从第j行选择，但符号相反（因为行指标不同会导致符号变化）。
    
    因此，所有的项都会两两抵消，最终det(A) = 0。
    
    另一种简单的证明方法：  
    如果第i行是第j行的k倍，我们可以进行初等行变换，将第i行减去第j行的k倍，得到一个新的矩阵B，其第i行全为0。根据行变换性质，det(A) = det(B)。而B有一行全为0，所以det(B) = 0，因此det(A) = 0。
    
3. 证明：  
    设A是n阶对称矩阵，则A^T = A
    
    我们可以利用矩阵的特征分解。由于A是对称矩阵，根据谱定理，A可以正交对角化：  
    A = PDP^T
    
    其中P是正交矩阵（即P^TP = I），D是对角矩阵，对角线上是A的特征值λ₁, λ₂, ..., λₙ
    
    考虑对角矩阵D的平方根矩阵D^(1/2)，它是一个对角矩阵，对角线上的元素是√λᵢ（如果λᵢ < 0，可以使用复数）
    
    令B = PD^(1/2)，则  
    $BB^T = PD^(1/2)(PD^(1/2))^T = PD^(1/2)(D^(1/2))^TP^T = PDP^T = A$
    
    因此，A可以分解为BB^T的形式。如果要求B是实矩阵，则需要A是半正定矩阵（所有特征值非负）。
    
4. 证明：  
    设A是n阶可逆矩阵，则A^(-1)存在，且AA^(-1) = A^(-1)A = I
    
    根据行列式的乘法性质，det(AA^(-1)) = det(A)·det(A^(-1))
    
    而det(AA^(-1)) = det(I) = 1
    
    因此，det(A)·det(A^(-1)) = 1，即det(A^(-1)) = 1/det(A)
    
5. 证明：  
    设A是n阶矩阵，且A^2 = 0
    
    假设A可逆，则存在A^(-1)使得AA^(-1) = A^(-1)A = I
    
    两边同时左乘A，得：  
    A^2A^(-1) = A
    
    由于A^2 = 0，所以：  
    0·A^(-1) = A  
    0 = A
    
    这与A可逆矛盾（可逆矩阵不能是零矩阵）。
    
    因此，A不可逆。
    

## 六、应用题答案

1. 解：  
    线性变换T将(x,y)映射到(2x-y, x+y)
    
    将这个变换表示为矩阵形式：  
    $T\begin{pmatrix} x \ y \end{pmatrix} = \begin{pmatrix} 2x-y \ x+y \end{pmatrix} = \begin{pmatrix} 2 & -1 \ 1 & 1 \end{pmatrix} \begin{pmatrix} x \ y \end{pmatrix}$
    
    因此，表示此线性变换的矩阵A = $\begin{pmatrix} 2 & -1 \ 1 & 1 \end{pmatrix}$
    
    判断是否可逆：  
    det(A) = 2×1 - (-1)×1 = 2 + 1 = 3 ≠ 0，所以A可逆
    
    求逆变换矩阵：  
    $A^{-1} = \frac{1}{det(A)} \begin{pmatrix} 1 & 1 \ -1 & 2 \end{pmatrix} = \frac{1}{3} \begin{pmatrix} 1 & 1 \ -1 & 2 \end{pmatrix} = \begin{pmatrix} \frac{1}{3} & \frac{1}{3} \ -\frac{1}{3} & \frac{2}{3} \end{pmatrix}$
    
    逆变换将(x,y)映射到$(\frac{x+y}{3}, -\frac{x-2y}{3})$
    
2. 解：  
    设线性变换由矩阵A表示，则需要满足：  
    $A\begin{pmatrix} 1 \ 1 \end{pmatrix} = \begin{pmatrix} 2 \ 1 \end{pmatrix}$，  
    $A\begin{pmatrix} 2 \ 3 \end{pmatrix} = \begin{pmatrix} 1 \ 4 \end{pmatrix}$，  
    $A\begin{pmatrix} 3 \ 2 \end{pmatrix} = \begin{pmatrix} 5 \ 1 \end{pmatrix}$
    
    检查这三个点是否共线：如果三个点共线，则只有两个线性无关的向量，无法唯一确定一个线性变换。
    
    计算向量$(2,3)-(1,1) = (1,2)$和$(3,2)-(1,1) = (2,1)$，这两个向量不共线（不成比例），所以三个点不共线。
    
    设$A = \begin{pmatrix} a & b \ c & d \end{pmatrix}$，代入条件：
    
    $\begin{pmatrix} a & b \ c & d \end{pmatrix} \begin{pmatrix} 1 \ 1 \end{pmatrix} = \begin{pmatrix} 2 \ 1 \end{pmatrix}$ 得 $a+b=2, c+d=1$
    
    $\begin{pmatrix} a & b \ c & d \end{pmatrix} \begin{pmatrix} 2 \ 3 \end{pmatrix} = \begin{pmatrix} 1 \ 4 \end{pmatrix}$ 得 $2a+3b=1, 2c+3d=4$
    
    由$a+b=2$和$2a+3b=1$，解得$a=5, b=-3$
    
    由$c+d=1$和$2c+3d=4$，解得$c=-1, d=2$
    
    验证第三个条件：  
    $\begin{pmatrix} 5 & -3 \ -1 & 2 \end{pmatrix} \begin{pmatrix} 3 \ 2 \end{pmatrix} = \begin{pmatrix} 15-6 \ -3+4 \end{pmatrix} = \begin{pmatrix} 9 \ 1 \end{pmatrix} \neq \begin{pmatrix} 5 \ 1 \end{pmatrix}$
    
    因此，不存在满足所有三个条件的线性变换矩阵A。
    
    事实上，线性变换必须保持原点不变，如果想要将三个任意点精确映射到三个目标点，通常需要使用仿射变换而非线性变换。
    
3. 解：  
    设Ax = b有唯一解，则A可逆，det(A) ≠ 0
    
    设A的第j列为a_j，则A = [a₁, a₂, ..., a_j, ..., a_n]
    
    将A的第j列替换为b得到的新矩阵B = [a₁, a₂, ..., b, ..., a_n]
    
    由于Ax = b有唯一解，设解向量为x = (x₁, x₂, ..., x_n)^T，则：  
    b = Ax = x₁a₁ + x₂a₂ + ... + x_ja_j + ... + x_na_n
    
    利用行列式的线性性质（对任一列进行线性组合，行列式也按相同方式线性组合），有：
    
    det(B) = det([a₁, a₂, ..., b, ..., a_n])  
    = det([a₁, a₂, ..., x₁a₁ + x₂a₂ + ... + x_ja_j + ... + x_na_n, ..., a_n])
    
    将b展开，利用行列式的多线性性质：
    
    det(B) = x₁det([a₁, a₂, ..., a₁, ..., a_n]) + x₂det([a₁, a₂, ..., a₂, ..., a_n]) + ... + x_jdet([a₁, a₂, ..., a_j, ..., a_n]) + ... + x_ndet([a₁, a₂, ..., a_n, ..., a_n])
    
    当矩阵中有两列相同时，行列式为零，所以上式中除了第j项外，其他项都为0：
    
    det(B) = x_jdet([a₁, a₂, ..., a_j, ..., a_n]) = x_jdet(A)
    
    因此，det(B) = x_j·det(A)，证毕。
