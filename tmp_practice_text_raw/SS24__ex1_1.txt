Recap - Vektoren, Matrizen und der Spann
1. Wir können jedes Lineare Gleichungssystem (LGS) schreiben als Ax = b wobei A eine
Matrix und b ein Vektor ist.
2. Das Matrix-Vektor Produkt Ax ist linear in x (Theorem 1.7.5).
3. Die Anzahl von Lösungen eines LGS ist entweder 0,1 oder unendlich (Theorem 1.3.1)
4. Über den Spann können wir den Lösungsraum eines LGS beschreiben (Theorem 1.7.6)
Fingerübungen
1. Wenn x eine Lösung für Ax = 0 ist und λ ∈ R, ist dann y = λx auch eine Lösung (d.h.
Ay = 0)? Gilt dasselbe auch für Ax = b (d.h. Ay = b)? (Theorem 1.7.5)
2. Wenn es nur endliche viele x1,...,xn gibt die das LGS Ax = 0 lösen, was ist dann n?
(Theorem 1.3.1 oder Fingerübung 1.)

(a) ∀x ∈ N ∃y ∈ N sodass x = y.
(b) ∃y ∈ N ∀x ∈ N sodass x = y.

Lösungsskizze zu Aufgabe 3
(a) Ausformuliert besagt die Aussage: “Für alle x in den natürlichen Zahlen existiert ein y in
den natürlichen Zahlen sodass x = y wahr ist.”
Diese Aussage ist wahr (setze y gleich y).
(b) Ausformuliert besagt die Aussage: “Es existiert ein y in den natürlichen Zahlen, sodass
für alle x in den natürlichen Zahlen x = y gilt.
Diese Aussage ist falsch.
(c) Ausformuliert besagt die Aussage: “Es existiert kein y in den natürlichen Zahlen, sodass
für alle x in den natürlichen Zahlen x = y gilt.
Diese Aussage ist wahr.

Hinweis: Wenn wir schreiben dass a ein Teiler von b ist, so meinen wir hier Teilen ohne Rest.
Lösungsskizze zu Aufgabe 4
1. Schritt 1: a,b,c ∈ N mit a teilt b und b teilt c ⇒ a teilt c.
Schritt 2: Eine Zahl d teilt eine Zahl e (ohne Rest) dann und genau dann, wenn ein
n ∈ N existiert sodass e = nd gilt.
Schritt 3: (direkter Beweis)
Da a teilt b und b teilt c, existieren m,n ∈ N sodass
b = ma und c = nb
gilt. Nun setzen wir b = ma in c = nb ein und erhalten a = n(ma). Da n,m ∈ N ⇒
n · m ∈ N. Daraus folgt:
∃k ∈ N (genauer k = mn) sodass c = ka.
⇒ a teilt c
2. Schritt 1: a,b ∈ N mit a + b ungerade ⇒ entweder a oder b ungerade.
Schritt 2: Eine Zahl k ∈ N ist ungerade wenn eine Zahl m ∈ N existiert sodass k = 2m+1.
Eine Zahl k ∈ N ist gerade wenn eine Zahl m ∈ N existiert sodass k = 2m.
Schritt 3: (Beweis durch Kontraposition)
Verneinung von “entweder a oder b ungerade”: “Entweder (a und b gerade) oder (a und
b ungerade)”.
Fall 1: (a und b ungerade)
⇒ ∃m,n ∈ N sodass a = 2m und b = 2n.
⇒ a + b = 2m + 2n = 2(m + n) und da m,n ∈ N ist auch m + n ∈ N. Daher ist a + b
wieder gerade.
Fall 2: (a und b ungerade)
⇒ ∃m,n ∈ N sodass a = 2m + 1 und b = 2n + 1.
⇒ a + b = (2m + 1) + (2n + 1) = 2(m + n + 1) und da m,n ∈ N ist auch m + n + 1 ∈ N.
Daher ist a + b wieder gerade.
Damit haben wir nun gezeigt, dass “Entweder (a und b gerade) oder (a und b ungerade)”
⇒ “a + b gerade” wahr ist. Und somit folgt die Behauptung über Kontraposition.
