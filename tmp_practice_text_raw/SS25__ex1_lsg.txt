Recap - Vektoren, Matrizen und der Spann
1. Wir können jedes Lineare Gleichungssystem (LGS) schreiben als Ax = b wobei A eine
Matrix und b ein Vektor ist.
2. Das Matrix-Vektor Produkt Ax ist linear in x (Theorem 1.7.5).
3. Die Anzahl von Lösungen eines LGS ist entweder 0,1 oder unendlich (Theorem 1.3.1)
4. Über den Spann können wir den Lösungsraum eines LGS beschreiben (Theorem 1.7.6)
Fingerübungen
1. Wenn x eine Lösung für Ax = 0 ist und λ ∈ R, ist dann y = λx auch eine Lösung (d.h.
Ay = 0)? Gilt dasselbe auch für Ax = b (d.h. Ay = b)? (Theorem 1.7.5)
2. Wenn es nur endlich viele x1,...,xn gibt, die das LGS Ax = 0 lösen, was ist dann n?
(Theorem 1.3.1 oder Fingerübung 1.)
Lösungsskizze zu Aufgabe 0
1. Für Ax = 0 gilt mit λ ∈ R laut Theorem 1.7.5.: A(λx) = λ(Ax) = λ0, somit ist auchλx
eine Lösung des LGS.
Für Ax = b gilt A(λx) = λb ̸= b im Allgemeinen, also ist λx i.A. keine Lösung.
2. Wenn es ein x ̸= 0 mit Ax = 0 gibt (wir nennen das eine nicht-triviale Lösung, die triviale
Lösung ist x = 0), dann ist λx laut 1. ebenfalls eine Lösung, somit gibt es unendlich viele
Lösungen, was ein Widerspruch zur Aufgabenstellung ist. Also muss x = 0 gelten, somit n = 1.
Alternative: Aus Theorem 1.3.1 wissen wir, dass jedes LGS 0,1 oder ∞ viele Lösungen hat,
somit kann es hier nur genau eine Lösung geben (denn x = 0 ist immer eine Lösung von
Ax = 0).

Aufgabe 2 (Spann)
Sei M = {m1,...,mk}, N = {n1,...,nl} ⊆ Rn
mit n,k,l ∈ N. Beweise oder widerlege die
folgenden Behauptungen.
(a) M ⊆ N ⇒ span(M) ⊆ span(N).
(b) span(M) ⊆ span(N) ⇒ M ⊆ N.
(c) span(M ∪ N) = span(M) ∪ span(N).
Lösungsskizze zu Aufgabe 2
Zu (a):
Die Aussage ist korrekt: Sei x ∈ span(M). Per Definition gilt dann, dass a1,...,ak ∈ R existieren
mit
x = a1m1 + ..akmk,
d.h. x ist eine Linearkombination von Vektoren aus M. Ein Vektor y ist genau dann in span(N),
wenn sich y als Linearkombination von Vektoren aus N schreiben lässt. Da M ⊆ N gilt
1,...,k ∈ N und somit x ∈ span(N).
Zu (b):
Die Aussage ist falsch: Es reicht ein Gegenbeispiel: Idee: 0 ist in jedem Spann enthalten - auch
wenn die zugrundeliegende Menge an Vektoren, die den Spann erzeugt, nicht die 0 enthält.
In anderen Worten, M = {0},N = {1} ⊂ R1
ist ein Gegenbeispiel, denn M ist nicht in N
enthalten aber span(M) = {0} ⊂ span(N) = R.
Alternative: für alle a ̸= b mit a,b,∈ R,a ̸= 0,b ̸= 0 gilt span{a} = span{b} = R, aber weder
{a} ⊆ {b} noch {b} ⊆ {a}.
Zu (c):
Die Aussage ist falsch: Es reicht wieder ein Gegenbeispiel: Sei
M = {e1},N = {e2} ⊂ R2
.
Dann ist span(M ∪N) = R2
, aber span(M) = {(x,0) : x ∈ R} und span(N) = {(0,x) : x ∈ R}
und somit span(M) ∪ span(N) = {(x1,x2) : x1 = 0 ∨ x2 = 0} ̸= R2
.

Aufgabe 3 (Quantoren)
Entscheide, ob die folgenden Aussagen wahr oder falsch sind
(a) ∀x ∈ N ∃y ∈ N sodass x = y.
(b) ∃y ∈ N ∀x ∈ N sodass x = y.
(c) ∄y ∈ N ∀x ∈ N sodass x = y.
Lösungsskizze zu Aufgabe 3
(a) Ausformuliert besagt die Aussage: “Für alle x in den natürlichen Zahlen existiert ein y in
den natürlichen Zahlen sodass x = y wahr ist.”
Diese Aussage ist wahr (setze y gleich y).
(b) Ausformuliert besagt die Aussage: “Es existiert ein y in den natürlichen Zahlen, sodass
für alle x in den natürlichen Zahlen x = y gilt.
Diese Aussage ist falsch.
(c) Ausformuliert besagt die Aussage: “Es existiert kein y in den natürlichen Zahlen, sodass
für alle x in den natürlichen Zahlen x = y gilt.
Diese Aussage ist wahr.

Schritt 2: Eine Zahl k ∈ N ist genau dann ungerade, wenn eine Zahl m ∈ N existiert

Schritt 3: (Beweis durch Kontraposition)
Wir zeigen: ¬ (entweder a oder b ungerade) ⇒ ¬ (a + b ungerade).
Verneinung von “entweder a oder b ungerade”: “Entweder (a und b gerade) oder (a und
b ungerade)”.
Fall 1: (a und b ungerade)
⇒ ∃m,n ∈ N sodass a = 2m und b = 2n.
⇒ a + b = 2m + 2n = 2(m + n) und da m,n ∈ N ist auch m + n ∈ N. Daher ist a + b
wieder gerade.
Fall 2: (a und b ungerade)
⇒ ∃m,n ∈ N sodass a = 2m + 1 und b = 2n + 1.
⇒ a + b = (2m + 1) + (2n + 1) = 2(m + n + 1) und da m,n ∈ N ist auch m + n + 1 ∈ N.
Daher ist a + b wieder gerade.
Damit haben wir nun gezeigt, dass “Entweder (a und b gerade) oder (a und b ungerade)”
⇒ “a + b gerade” wahr ist. Und somit folgt die Behauptung über Kontraposition.
