Recap - Spaltenraum und Kern
  1. Kern und Spaltenraum sind wichtige Größen einer n × m-Matrix: ihre Dimensionen sum-
     mieren sich zur Dimension des Startraums m = dim(ker(A))+rang(A). (Theorem 3.4.12)

  2. Damit ist die Invertierbarkeit einer Matrix A äquivalent dazu, dass A vollen Rang (rang(A) =
     n) bzw. einen trivialen Kern (ker(A) = {0}) hat. (Theorem 3.4.13)


Fingerübungen
  1. Sei A eine n × m-Matrix mit ker(A) = {0}. Was ist der Rang rang(A) von A? (Theorem
     3.4.12)

  2. Sei A eine n×m-Matrix mit ker(A) = Rm . Was ist der Spaltenraum col(A)? (Nachdenken)

  3. Kann es eine 2023 × 2024-Matrix A geben mit ker(A) = {0}? (Theorem 3.4.12)

Aufgabe 4 (Funktionenraum ist Vektorraum)
Zeige, dass der Raum aller Funktionen mit der in Beispiel 3.1.5 definierten Addition und Ska-
larmultiplikation ein Vektorraum ist. Beweise hierfür die Eigenschaften aus Definition 3.1.1.


Lösungsskizze zu Aufgabe 4
Sei im folgenden F der Funktionenraum
                                        F := {f : R → R}.
Dieser Raum ist nicht leer, da z.b. N : R → R, x 7→ 0 ein Element aus F ist. Ein Vektorraum
benötigt eine Addition und eine Multiplikation, die wie folgt definiert sind:
   1. Sei f, g ∈ F dann setzen wir (f + g) : R → R, x 7→ f (x) + g(x).
   2. Sei f ∈ F und c ∈ R, dann definieren wir cf : R → R, x 7→ cf (x).
Nun müssen wir alle Eigenschaften der der Definition 3.1.1 durchgehen: Seien dazu f, g, h ∈ F
und d, c ∈ R.
  (i) (f + g) definiert wieder eine Funktion zwischen R und R und somit ist f + g ∈ F.
 (ii) Für alle x ∈ R gilt (R ist kommutativ):
                         (f + g)(x) = f (x) + g(x) = g(x) + f (x) = (g + f )(x).
      Daher gilt f + g = g + f .
 (iii) Für alle x ∈ R gilt (R ist assoziativ):
                      ((f + g) + h)(x) = (f + g)(x) + h(x) = f (x) + g(x) + h(x)
                       = f (x) + (g + h)(x) = (f + (g + h))(x).

 (iv) Die Funktion N von oben entspricht dem Nullvektor. Da für alle x ∈ R gilt
                             (f + N )(x) = f (x) + N (x) = f (x) + 0 = f (x).

  (v) Sei f ∈ F. Dann ist −f ∈ F (ist wieder eine Funktion zwischen R und R) und es gilt für
      alle x ∈ R:
                     (f + (−f ))(x) = f (x) + (−f (x)) = f (x) − f (x) = 0 = N (0).

 (vi) cf definiert wieder eine Funktion zwischen R und R und somit ist cf ∈ F.
(vii) Für alle x ∈ R gilt
                             (c(f + g))(x) = cf (x) + cg(x) = (cf + cg)(x)

(viii) Für alle x ∈ R gilt
          ((c + d)f )(x) = (c + d)f (x) = cf (x) + df (x) = (cf )(x) + (df )(x) = (cf + df )(x).

 (ix) Für alle x ∈ R gilt
                             (c(df ))(x) = cdf (x) = (cd)f (x) = ((cd)f )(x).

  (x) Für alle x ∈ R gilt
                                        (1f )(x) = 1f (x) = f (x).

Damit folgt mit Definition 3.1.1, dass F mit der oben definierten Addition und Skalarmulitpli-
kation ein Vektorraum ist.
