Recap - Lineare Unabhängigkeit und Abbildungen
  1. Haben wir eine Lösung x0 mit Ax0 = b gefunden, dann lässt sich jede weitere Lösung
     schreiben als x0 + x wobei Ax = 0 gilt. Das heißt, um Ax = b zu studieren reicht es
     b = 0 anzunehmen. (Theorem 1.8.6)

  2. Die Menge aller Ax = 0 trägt eine lineare Struktur. Das heißt wenn Ax1 = Ax2 = 0 gilt
     so ist auch A(ax1 + bx2 ) = 0. Es gilt sogar allgemeiner...

  3. ...jede Lösung lässt sich als Linearkombination von endlich vielen und fixen xi schreiben.
     Mathematisch ausgedrückt, {x : Ax = 0} = span{x1 , ..., xn }. (Theorem 1.8.3)

  4. Lineare Unabhängigkeit wird das Kernkonzept um die Anzahl n der Vektoren im Spann
     span{x1 , ..., xn } möglichst gering zu halten. (Theorem 1.9.2 und Aufgabe 2.)

  5. Die charakterisierende Eigenschaft von LGS, d.h. von Matrizen, ist, dass sie lineare Ab-
     bildungen definieren. Mathematisch ausgedrückt: jede Matrix definiert eine (eindeutige)
     lineare Abbildung und umgekehrt. (Theorem 1.10.2)

  6. In anderen Worten: das Studium von LGS ist das Studium von linearen Abbildungen.


Fingerübungen
  1. Kann es 2024 linear unabhängige Vektoren im R2023 geben? (Theorem 1.9.5)

  2. Wenn zwei lineare Abbildungen dieselbe darstellende Matrix haben, sind sie dann gleich?
     (Theorem 1.10.2)

  3. Wenn die Matrix A eine lineare Abbildung f darstellt und Ax = 0 gilt, was ist dann
     f (x)?

Aufgabe 1 (Die Menge der Linearen Abbildungen ist linear)
Seien f, g : Rn → Rm lineare Abbildungen und a, b ∈ R. Zeige, dass dann auch

                                         h = af + bg : Rn → Rm

definiert durch h(x) = af (x) + bg(x) linear ist.


Lösungsskizze zu Aufgabe 1
Zu zeigen: h(x + y) = h(x) + h(y) und h(c x) = c h(x) für alle x, y ∈ Rn , c ∈ R. Es gilt
              Def. von h                                f, g linear
   h(x + y)     =     af (x + y) + bg(x + y) = a(f (x) + f (y) + b(g(x) + g(y))
              = af (x) + bg(x) + af (y) + bg(y) = h(x) + h(y),
              Def. von h                      f, g linear
      h(cx)      =         af (cx) + bg(cx)       =         acf (x) + bcg(x) = c(af (x) + bg(x)) = c h(x).

Somit ist h eine lineare Abbildung.

Hinweis: Theorem 1.9.2.


Lösungsskizze zu Aufgabe 2
Idee: wenn die Menge {x1 , ..., xk } linear abhängig ist, dann können wir einen Vektor daraus ent-
fernen, ohne den Spann zu verändern (folgt aus Th. 1.9.2). Das kann man so lange wiederholen,
bis die resultierende Menge linear unabhängig ist. Konkreter: Wir starten bei x1 und entfernen
es aus der Menge, wenn sich dadurch der Spann nicht verändert. Aus der resultierenden Menge
entfernen wir x2 , wenn sich dadurch der Spann nicht verändert etc.
Beweis: Definiere U0 , U1 , . . . , Uk wie folgt:

                                                 i−1 \ {xi }   wenn xi ∈ span(Ui−1 \ {xi })

Behauptung: Uk ist linear unabhängig und span(Uk ) = span(U0 ), somit ist Uk die oben gesuchte
Menge {xi1 , ..., xil }.

   • span(Uk ) = span(U0 ): Aus der Definition von Ui folgt, dass span(Ui ) = span(Ui−1 ), da
     entweder Ui = Ui−1 oder xi ∈ span(Ui−1 \{xi }) und somit span(Ui−1 \{xi }) = span(Ui−1 ).
     Also folgt induktiv span(Uk ) = span(U0 ).

   • Uk ist linear unabhängig: Beweis durch Widerspruch: Sei Uk linear abhängig, d.h. es
     gibt ein xi ∈ Uk , sodass xi ∈ span(Uk \ {xi }). Dann gilt xi ∈ Ui (da Uk ⊆ Ui ) und xi ∈
     span(Ui−1 \{xi }) (da Uk \{xi } ⊆ Ui−1 \{xi } und somit span(Uk \{xi }) ⊆ span(Ui−1 \{xi }),
     siehe Übungsblatt 1). Dann hätten wir aber Ui als Ui−1 \ {xi } gewählt und somit xi ∈ / Ui
     → Widerspruch.

    Aus Fall 1 und Fall 2 folgt nun die Behauptung.
    Alternativ funktioniert das ganze auch ohne Verwendung des Spanns: Sei x ∈ L. Dann
    ist cx (c ̸= 0) ebenfalls eine Lösung des LGS. Also ist cx ∈ L und somit cL ⊆ L. Noch
    zu zeigen: L ⊆ cL. Sei x ∈ L. Dann ist auch 1/cx (c ̸= 0) eine Lösung des LGS. Da also
    1/cx ∈ L, folgt x ∈ cL.

  2. Zu zeigen: x1 + L = x2 + L

    Fall 1: L = {0}
    Beweis durch Widerspruch. Nehme an, dass x1 ̸= x2 . Dann gilt

                                  Ax1 = b = Ax2 und x1 − x2 ̸= 0
                            ⇒ A(x1 − x2 ) = Ax1 − Ax2 = b − b = 0
                                                ⇒ 0 ̸= x1 − x2 ∈ L

    Das ist aber ein Widerspruch zu L = {0} ⇒ x1 = x2 und Ax = b hat hier nur genau
    eine Lösung.

    Fall 2: L ⊋ {0}.

    Wir zeigen zuerst, dass x1 + L ⊆ x2 + L. Sei x ∈ x1 + L, d.h. es existiert ein y ∈ L,
    sodass x = x1 + y. Zu zeigen: x ∈ x2 + L, d.h. es existiert ein ỹ ∈ L, sodass x = x2 + ỹ.
    Es gilt
                     x = x1 + x2 − x2 + y = x2 + (x1 − x2 + y) = x2 + ỹ
    mit ỹ = x1 − x2 + y. Es gilt ỹ ∈ L, da

                       A(x1 − x2 + y) = Ax1 − Ax2 + Ay = b − b − 0 = 0.

    Alternativ kann man ähnlich zeigen, dass x1 ∈ x2 + L (da x1 = x2 + (x1 − x2 )), woraus
    x1 + L ⊆ x2 + L folgt.

    Die andere Inklusion: x1 + L ⊇ x2 + L folgt analog.

    Aus Fall 1 und Fall 2 folgt nun die Behauptung.
