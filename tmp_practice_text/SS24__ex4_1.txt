Recap - Invertierbarkeit von Matrizen
  1. Das Matrix Produkt verhält sich ähnlich wie das Produkt von Zahlen (Theorem 2.1.8)
     unterscheidet sich aber grundlegend in zwei Eigenschaften: es ist nicht kommutativ
     (d.h. AB 6= BA i.A.) und besitzt kein Inverses (d.h. @B s.d. AB = 1 i.A.)

  2. Dass ein Inverses (d.h. A−1 ) existiert ist (unter Anderem) äquivalent zur eindeutigen
     Lösbarkeit des LGS Ax = b für beliebige b (Theorem 2.2.15)

  3. Schränken wir uns auf invertierbare Matrizen ein, so gelten analoge Rechenregeln wie
     (AB)−1 = B −1 A−1 (Theorem 2.2.3)


Fingerübungen
  1. Wenn A, B invertierbare Matrizen sind, ist dann auch AB invertierbar? (Theorem 2.2.3)

  2. Wenn A eine invertierbare Matrix ist, ist dann auch AT invertierbar? (Theorem 2.2.3)

  3. Wenn das LGS Ax = b eine Lösung besitzt und B eine invertierbare Matrix ist, besitzt
     dann das LGS (AB)x = b eine Lösung? (Theorem 2.2.15)


Übungsaufgaben
Aufgabe 1 (Das Inverse einer linearen Abbildung ist linear)
Sei f eine lineare Abbildung. Zeige: Wenn f eine invertierbare Abbildung ist, d.h. es existiert
eine Abbildung f −1 mit f ◦ f −1 (x) = x für alle x, dann ist f −1 linear.

Aufgabe 2 (Invertierbarkeit und bijektive Abbildungen)
Sei f : Rn → Rn eine lineare Abbildung.

  • Wir nennen f injektiv, falls für alle x ∈ Rn mit f (x) = 0 schon x = 0 folgt.

  • Wir nennen, f surjektiv falls für alle y ∈ Rn ein x ∈ Rn existiert mit f (x) = y.

Zeige: f ist invertierbar genau dann wenn f injektiv ist genau dann wenn f surjektiv ist.

Hinweis: Nutze die Darstellung von f über eine Matrix A. Was bedeutet es für A wenn f
injektiv bzw. surjektiv ist? Nutze dann Theorem 2.2.15.
